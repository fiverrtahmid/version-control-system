Only time for new repo

git init
git add README.md (replace README.md with your file name or keep it if you want to add the readme.md file or replace it with the dot(.) for adding all the files to the repo )
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/fiverrtahmid/version-control-system.git
git push -u origin main

Then 3 step for the existing repo

  1. git add .
  2. git commit -m"comment here"
  3. git push
